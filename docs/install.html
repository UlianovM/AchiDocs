Синхронизация

#

Статус синхронизации

#

Подготовка диска

Информация

Отобразить информацию об ОЗУ

#
free -gh

Отобразить все блочные устройства (диски)

#
lsblk

Больше информации (тип файловой системы и точка монтирования)

#
lsblk -fp

Разметка

Клиент fdisk

#
fdisk /dev/disk

Форматирование

Форматирование boot раздела в FAT32

#
mkfs.fat -F32 /dev/boot_раздел -n label

Форматирование root раздела в EXT4

#
mkfs.ext4 /dev/root_раздел -L label

Форматирование swap раздела

#
mkswap /dev/swap_раздел -L label

Монтирование

Монтирование root раздела по пути /

#
mount /dev/root_раздел /mnt

Монтирование boot раздела по пути /boot

#
mount /dev/boot_раздел /mnt/boot --mkdir

Монтирование (активация) swap

#
swapon /dev/swap_раздел

Установка

Поиск зеркал

Поиск самых быстрых зеркал в России и сохранение 8 из них

#
reflector -c Russia \ -p https \ -n 8 \ --sort rate \ --save /etc/pacman.d/mirrorlist \ --download-timeout 1 \ --connection-timeout 1

Скачивание

Скачивание и установка пакетов в новую систему

#
pacstrap -P /mnt base base-devel linux \ linux-firmware amd-ucode intel-ucode \ iwd networkmanager nano \ less sudo reflector \ bash zsh fish \ man-db man-pages texinfo

Первичная настройка

Сохранение точек монтирования

#
genfstab -U /mnt > /mnt/etc/fstab

Частичный перехо в новую систему

#
arch-chroot /mnt /bin/shell
