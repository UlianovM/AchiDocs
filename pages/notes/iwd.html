iwd

Инструмент для настройки демона iwd через интерфейс D-Bus. Поддерживает как интерактивный режим, так и режим командной строки.

Интерактивный режим

Вход в интерактивный режим:

#
iwctl

Таблица доступных интерфейсов:

[iwd]#
device list

Сканирование сети:

[iwd]#
station device scan

device - имя интерфейса, например wlan0


Таблица доступных сетей:

[iwd]#
station device get-networks

В интерактивном режиме таблица обновляется автоматически


Подключение:

[iwd]#
station device connect SSID

SSID - имя сети

Для подключения к некоторым сетям необходимо иметь пожддержку русских символов

Пробелы можно указать с помощью escape символа \

Пример: station device connect Net\ Work)


Выход из интерактивного режима:

[iwd]#
exit

Режим коммандной строки

Все теже комманды, но с указанием каталога iwctl, полезно автоматизации / создания скриптов

Пример:

#
iwctl station device connect SSID
