fdisk

это диалоговая программа для создания таблиц разделов и управления ими

Вход в клиент утилиты:

#
fdisk [options] disk

Отобразить таблицу разделов диска:

#
fdisk -l disk

Интерактивный режим

Комманды

wwriteЗаписать изменения на диск
qquitВыйти без применения изменений
ggptСоздать чистую GPT разметку (для всего диска)
odosСоздать чистую MBR разметку (для всего диска)
nnewСоздать новый раздел
ttypeЗадать тип разделу
llistОтобразить всё допустимые типы разделов (и их алиасы)
eeditИзменить размер раздела
TTrimНе используется
pprintОтобразить таблицу разделов (изменённую)
FFreeОтобразить нераспределённое место
ddeleteУдалить раздел

Часто используемые алиасы типов разделов

1EFI System
4BIOS Boot
19Linux swap
20Linux filesystem
22Linux root (x86)
23Linux root (x86_64)
42Linux home
47Linux usr (x86)
48Linux usr (x86_64)

Параметры fdisk

-b sectorsize, --sector-size sectorsize

Размер сектора диска, допустимые значения - 512, 1024, 2048 и 4096

-L, --color[=when]

Цветной вывод, необязательным аргументом when может быть значение auto, never или always, по умолчанию auto

-l, --list

Выведит список таблиц разделов для указанных устройств, а затем завершит работу. Если устройства не указаны, используются устройства, указанные в /proc/partitions (если этот файл существует). Устройства всегда перечисляются в том порядке, в котором они указаны в командной строке или ядром, указанным в /proc/partitions.

-x, --list-details

Как и --list, но содержит более подробную информацию

-n, --noauto-pt

Автоматически не создаёт таблицу разделов по умолчанию на пустом устройстве. Таблица разделов должна быть явно создана пользователем (с помощью команд o, g и т.д.).

-o list, --output list

Указывает, какие выходные столбцы печатать. Используйте --help, чтобы получить список всех поддерживаемых столбцов. Список столбцов по умолчанию может быть расширен, если список указан в формате +list (например, -o +UUID)

-w [when], --wipe when

Удаляет подписи файловой системы, RAID и таблицы разделов с устройства, чтобы избежать возможных коллизий. Параметром when может иметь значения auto, never или always. Если этот параметр не задан, по умолчанию используется значение auto, и в этом случае подписи удаляются только в интерактивном режиме. Во всех случаях обнаруженные сигнатуры отображаются предупреждающими сообщениями перед созданием новой таблицы разделов

-W [when], --wipe-partitions when

Удаляет подписи файловой системы, RAID и таблицы разделов из только что созданных разделов, чтобы избежать возможных коллизий. Аргументом when может быть значение auto, never или always. Если этот агрумент не указан, по умолчанию используется значение auto, и в этом случае подписи удаляются только в интерактивном режиме и после подтверждения пользователем. Во всех случаях обнаруженные подписи отображаются предупреждающими сообщениями перед созданием нового раздела

-h, --help

Выведет help документацию

-V, --version

Отобразит версию

Некоторые специфичные флаги здесь не указаны
